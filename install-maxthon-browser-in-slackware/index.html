在Slackware上安装Maxthon浏览器

linux Nov 08, 2016 我勒个去

近期把自己台式机的deepin linux发行版给格式化了,因为更新之后进入不了桌面,然后又莫名其妙的一大堆问题,于是下载了个Slackware的DVD进行安装。
不得不说,Slackware的系统清爽很多,虽然不支持搜狗输入法,但是使用fcitx自带的中文输入法已经绰绰有余了,而且还很快。
由于平时使用Maxthon浏览器看新闻比较多,而默认情况下Slackware没有提供对应的版本,因此只能自己动手丰衣足食了。
由于近期Maxthon页面的改版,看不到对应的下载链接,而新的页面是关于Maxthon 5的,也没有提供Linux的版本。
在这里我们下载最新的Linux版本,从源码进行操作,下面是稳定版本的链接:

stable64
stable32

点击下载之后,我们进行解压操作: 

  
dog@darkstar:~/Downloads$ tar xvf maxthon-browser-stable-1.0.5.3-x86_64.tgz   
maxthon/  
maxthon/product_logo_22.png  
maxthon/nacl_helper  
maxthon/conf.d/  
maxthon/conf.d/pn  
maxthon/conf.d/version  
maxthon/MxEncode.so  
maxthon/chrome_100_percent.pak  
maxthon/product_logo_256.png  
maxthon/maxthon  
maxthon/maxthon.desktop  
maxthon/nacl_helper_bootstrap  
maxthon/content_resources.pak  
...

解压之后,我们进入该目录中运行: 

  
dog@darkstar:~/Downloads$ cd maxthon  
dog@darkstar:~/Downloads/maxthon$ ./maxthon

它会寻找/opt/maxthon/maxthon_sandboax可执行文件。为了能顺利的调用该浏览器,我们需要安装旧版本的libgcrypt,我们可以通过下载如下的2个文件:

  • libgcrypt包
  • libgcrypt15安装脚本

然后我们使用上面的方式解压libgcrypt15.tar.gz压缩包,然后将libgcrypt-1.5.4.tar.bz2放在解压后的文件夹中。
之后我们进入上述的解压后的目录并执行: 

  
dog@darkstar:~/Downloads/libgcrypt15$ sudo ./libgcrypt15.SlackBuild

此时SlackBuild的构建脚本会自动执行并进行编译操作,最终生成Maxthon浏览器需要的libgcrypt.so.11共享库文件。
最后我们将Maxthon的目录下的内容复制到/opt/maxthon盘符下,之后点击其桌面文件即可出现如下的页面了:

然后我们就可以欢快的使用Maxthon浏览器了。
