简述图数据库OrientDB

database Dec 23, 2017 我勒个去

自从项目引入neo4j这个图数据库之后,但苦于导入数据问题实在太坑爹,需要手动进行操作,于是就搜索了其他的数据库。
关于OrientDB,可以解决这个数据导入的问题,但是对内存的要求就高了,至少需要2G内存才能正常运行。
我们可以通过下面的链接下载历史的版本: 

  
http://orientdb.com/download-previous/

而在Centos 7上进行安装的方式可以参考。其他的安装方式可以参考。
关于orientdb与neo4j的对比可以参考如下的链接: 

  
http://orientdb.com/orientdb-vs-neo4j/

与其他数据库的比较可以参考如下的链接: 

  
https://www.arangodb.com/2015/10/benchmark-postgresql-mongodb-arangodb/

而从关系型数据库导入数据的方式参考如下的链接: 

  
http://orientdb.com/docs/2.2/Import-from-DBMS.html
