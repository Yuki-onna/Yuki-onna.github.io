在线音乐API推荐

other Mar 13, 2017 我勒个去

关于音乐API的问题,其实源自近期自己的1个小需求,使用Python编写1个在线音乐服务器。在这里,使用Python将音乐播放部分的代码给实现了,但是没有找到比较好的在线音乐API。接着,求助于之前的某哥,他给我推荐2个API,1个是网易的,另外1个是百度的。

网易API

对于网易的API,主要有如下一些参数:

参数 取值
src lofter,可为空
type 1
s 查询的关键字
limit 返回的结果条数
offset 偏移
callback 回调,为空则返回json,否则为jsonp

在这里我搜索关键字王菲: 

http://s.music.163.com/search/get/?type=1&limit=10&s=王菲

百度音乐API

http://tingapi.ting.baidu.com/v1/restserver/ting?from=webapp_music&method=baidu.ting.search.catalogSug&format=json&callback=&query=王菲&_=1413017198449
