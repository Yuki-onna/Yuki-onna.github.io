《WireShark网络分析实战》读后感

network wireshark Sep 09, 2016 我勒个去

这本《WireShark网络分析实战》的书是在今年亚马逊200块减50块时购买的,之前没怎么看。近期由于在项目中需要优化APP,特意拿来翻阅了下。
看完的感觉就是我后悔了,不是内容有多么的难,而是没什么收获。
为什么这样感觉呢,因为感觉作者很多东西讲的不清不楚的,完全就是东拼西凑组合在一起。虽然书的名字是网络分析实战,但是完全是在讲基础的一些东西,完全没有什么实战性。
当初就是看着亚马逊中评论才买的,拿到手后翻阅后就这个感觉。
虽然本书的作者有30多年的网络经验,但是这样写法真心不怎么能体会其经验的丰富。突然感觉在IT行业,经验不是由工作年龄来计算的,而是看你的学习能力多强,经历多少失败得出的。
之前工作的公司请了1个技术顾问,虽然做了20多年的Java开发,但是感觉能力上不还如我,当然人家能当技术顾问,自然有他的过人之处。当然,那家公司都不怎么懂技术,也就那样。
不管如果希望学习网络的话,还是推荐如下的2本书:

  • 谢希仁的《计算机网络》第6版
  • 林沛满的《WireShark网络分析就这么简单》和《WireShark网络分析的艺术》

当然搞技术不要在于技术书很贵,有些书花那个价格还是值得的。
当然,搞技术如果只看技术书还是远远不够的,有时要看一些其他的书籍,比如经济学、数学、心理学方面的书籍。
